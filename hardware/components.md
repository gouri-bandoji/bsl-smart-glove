# Hardware Components

## Microcontroller
Arduino Nano 33 IoT

## Sensors
- Flex Sensors (finger bending detection)
- Capacitive Touch Sensors (finger contact detection)
- LSM6DS3 IMU (accelerometer and gyroscope)

## Communication
WiFi UDP communication for real-time data transmission.

## Power
Portable battery supply for wearable operation.

## System Purpose
To collect real-time gesture data for British Sign Language (BSL) recognition.
