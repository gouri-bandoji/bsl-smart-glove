# BSL Smart Glove – Gesture Recognition System

This project presents a wearable smart glove designed to detect and interpret British Sign Language (BSL) gestures using multiple sensors and wireless communication.

The system captures hand movements using flex sensors, capacitive touch sensors and an IMU, then transmits the data via WiFi to a computer for processing and logging.

## Technologies Used

- Arduino Nano 33 IoT
- Embedded C/C++
- Python
- UDP Communication
- WiFi Networking
- Sensor Integration

## Hardware Components

- 10 Flex Sensors
- Capacitive Touch Sensors
- LSM6DS3 IMU
- Arduino Nano 33 IoT
- Wireless Communication

## System Architecture

Sensors → Arduino Nano 33 IoT → WiFi UDP → Python Receiver → Dataset Logging

## Features

- Real-time gesture data acquisition
- Wireless sensor data transmission
- Multi-sensor fusion
- Data logging for gesture recognition datasets
